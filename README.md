# i2c-swift-example

This repository demonstrates how to control I2C devie using Swift. 

## Supported Platform

* I2CTinyUSB adapter (Mac and Linux)
* I2C Kernel device (/dev/i2c*) (Respberry Pi etc...)

## Building


### macOS

```
$ brew install libusb-compat libusb
```

### Linux(Ubuntu 16.04)


```
$ sudo apt-get install 
```
